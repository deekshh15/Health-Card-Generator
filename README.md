# Health Tracker System (C++)

A console-based Health Tracker System developed using C++ and OOP concepts.

## Features
- Multiple user health cards
- BMI, BMR, and calorie analysis
- Lifestyle & eating habit tracking
- Weekly adherence report with progress bar
- Update height & weight
- Export health card to file

## Technologies Used
- C++
- Object-Oriented Programming
- STL (vector)
- File handling

## How to Run
```bash
g++ menu.cpp -o health
./health
